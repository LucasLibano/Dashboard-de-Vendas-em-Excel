Dashboard de Vendas

Excel Avançado • Tabelas Dinâmicas • KPIs

- Sobre o projeto

Neste projeto, eu construí um **dashboard de vendas interativo e profissional no Excel**, utilizando uma base real de dados de 2024 com mais de **1 milhão de linhas**.

O meu objetivo foi transformar dados brutos em informações estratégicas para análise de desempenho comercial.

- Tratamento de dados

Antes de montar o dashboard, eu realizei a etapa de limpeza e preparação da base:

 Removi linhas em branco
 Corrigi números que estavam armazenados como texto
 Padronizei e organizei os dados para análise

- Métricas criadas

Durante o projeto, criei métricas importantes para análise de performance:

- **Faturamento total**
- **Meta de vendas**


- Análises do dashboard

Com o dashboard, consigo analisar:

Evolução das vendas mensais
Comparativo entre **Meta vs Faturamento**
Ranking de produtos
Top clientes por faturamento
Produtos com melhor desempenho

- Insights gerados

Com essa análise, consigo:

Identificar os produtos mais lucrativos
Entender quais clientes são mais importantes
Acompanhar a performance mensal de vendas
Comparar resultados reais vs metas estabelecidas

- Ferramentas utilizadas

Microsoft Excel
Tabelas Dinâmicas
Fórmulas e funções avançadas
Tratamento e modelagem de dados

- Resultado final

Consegui transformar uma base grande e complexa em um **dashboard dinâmico e visual**, que gera insights claros para tomada de decisão.
